<h2>프로젝트 영상</h2>

[![Video Label](http://img.youtube.com/vi/H5vfu0tG8lY/0.jpg)](https://youtu.be/H5vfu0tG8lY)

<h2>프로젝트 개요</h2>

<p align="center"><img width="550" alt="1" src="https://github.com/artist1996/CSM_WinAPI/assets/160116004/9de06537-d0fe-4cf3-9d92-3e49daea93e3"></p>

```
1997년 8월 1일 출시된 Capcom사의 RockMan X4의 모작으로, C++, WinAPI를 사용해 제작 했습니다.
개발자는 Animation Tool 및 Editor를 개발해 Sprite이미지를 활용한 Animation 제작 및 Editor를 활용한
Line과 Platform 을 활용한 지형 및 몬스터 및 장애물 Object를 해당 위치에 직접 배치 할 수 있습니다.
```

<h2>프로젝트 설명</h2>

<h4>게임 설명</h4>

<p align="center"><img width="550" alt="1" src="https://github.com/artist1996/CSM_WinAPI/assets/160116004/f3411fce-a71b-4f4b-bf5c-af90adda111c"></p>

<h4>프로젝트에 반영된 게임의 특징</h4>

* 게임 플레이 및 매커니즘
```
RockMan X4는 좌/우/점프/벽타기/벽차기
등으로 이루어진 2D 횡스크롤 게임 입니다.

1인용 게임으로 Stage의 몬스터를 제거하고
각 Stage의 보스를 클리어하는 방식의
게임입니다.
```

* 스테이지
```
원작 게임을 고증하여 게임 시작 시
TITLE, Chracter Select, Loading, Stage01, Stage02 로 구성 했습니다.
Stage01에서 문을 파괴하면 Stage02로 진입하며 Stage02에선 보스전을 진행합니다.
```

* 캐릭터
```
게임 플레이에 사용되는 캐릭터는「제로」입니다.
```
* 몬스터 및 보스
```
4가지의 몬스터와 3가지의 장애물 Object 그리고 보스를 구현했습니다.
보스는 총 4가지 패턴을 사용합니다.
```
